# Векторный редактор (CLI)

Простой векторный редактор с интерфейсом командной строки. Позволяет создавать геометрические фигуры (точку, отрезок, окружность, квадрат), просматривать список созданных фигур и удалять их. Фигуры хранятся в оперативной памяти с присвоением уникального идентификатора (ID).

## Функциональность

- **Создание фигур**: точка, линия (отрезок), окружность, квадрат.
- **Просмотр списка фигур**: вывод всех созданных фигур с их ID и характеристиками.
- **Удаление фигуры** по ID.
- **Справка** по командам.
- **Выход** из программы.

## Установка и запуск

### Требования
- Python 3.7 или выше
- Git (для клонирования репозитория)

1. Клонируйте репозиторий:
   ```bash

   cd vector-editor
   ```

2. Запустите программу:
   ```bash
   python main.py
   ```

## Использование

После запуска отображается приветствие и список доступных команд. Далее вводите команды в интерактивном режиме.

### Команды

| Команда | Описание |
|---------|----------|
| `add`   | Перейти в режим создания фигуры. Будет предложено выбрать тип фигуры и ввести координаты. |
| `list`  | Вывести список всех фигур с их ID и характеристиками. |
| `delete`| Удалить фигуру по ID. После ввода команды нужно указать ID. |
| `help`  | Показать список команд. |
| `exit`  | Завершить работу программы. |

### Создание фигур

При вводе команды `add` программа последовательно запрашивает тип фигуры и параметры:

- **Точка**: координаты `x` и `y` (целые числа).
  ```
  Введите координаты точки x и y через пробел: 10 20
  ```

- **Линия**: координаты начала и конца `x1 y1 x2 y2`.
  ```
  Введите координаты линии x1 y1 x2 y2 через пробел: 0 0 5 5
  ```

- **Окружность**: координаты центра `x y` и радиус `r`.
  ```
  Введите координаты центра окружности и радиуса x y r через пробел: 0 0 10
  ```

- **Квадрат**: координаты четырёх вершин в порядке обхода `x1 y1 x2 y2 x3 y3 x4 y4`.
  ```
  Введите координаты вершин квадрата x1 y1 x2 y2 x3 y3 x4 y4 через пробел: 0 0 0 5 5 5 5 0
  ```
  Программа проверяет, что введённые точки действительно образуют квадрат (равенство сторон и диагоналей). Если проверка не пройдена, выбрасывается исключение, и фигура не создаётся.

Для выхода из режима создания без добавления фигуры введите `Возврат` (регистр не важен).

## Зависимости

Проект использует только стандартную библиотеку Python (модули `math` для вычислений, `sys` не используется явно). Дополнительных пакетов не требуется.

## Примечания

- При вычислении длины отрезка используется метод `__len__`, который возвращает число с плавающей точкой. Это допустимо только при прямом вызове метода (например, `line.__len__()`), но не рекомендуется использовать со встроенной функцией `len()`, так как последняя ожидает целочисленный результат. В текущей реализации везде используется `__len__()` как метод, поэтому ошибок не возникает.
- Для проверки корректности квадрата используется точное сравнение длин сторон и диагоналей. Из-за погрешностей вычислений с плавающей точкой возможно, что идеальный квадрат с целочисленными координатами будет распознан верно, но для более сложных случаев рекомендуется использовать `math.isclose()`.
- Все координаты и радиус вводятся как целые числа (тип `int`). При необходимости можно модифицировать код для поддержки вещественных чисел.
