# Векторный редактор (CLI) v0.02

Простой векторный редактор с интерфейсом командной строки. Позволяет создавать геометрические фигуры, просматривать список созданных фигур, удалять их, а также сохранять и загружать данные в JSON-файлы.

## Возможности

### Поддерживаемые фигуры:
- **Точка** (point) — задаётся координатами x, y
- **Линия** (line) — задаётся двумя точками (x1, y1, x2, y2)
- **Окружность** (circle) — задаётся центром (x, y) и радиусом
- **Квадрат** (square) — задаётся четырьмя вершинами (x1, y1, x2, y2, x3, y3, x4, y4)
- **Овал** (oval) — задаётся центром (x, y) и двумя радиусами (полуосями)
- **Прямоугольник** (rectangle) — задаётся четырьмя вершинами (x1, y1, x2, y2, x3, y3, x4, y4)

### Функции:
- ✅ Создание фигур через командную строку
- ✅ Просмотр списка всех созданных фигур с их ID
- ✅ Удаление фигур по ID
- ✅ Сохранение списка фигур в JSON-файл
- ✅ Загрузка списка фигур из JSON-файла
- ✅ Валидация корректности фигур при создании (для квадрата и прямоугольника)

## Установка и запуск

### Требования
- Python 3.7 или выше
- Git (для клонирования репозитория)

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Gelt17/vector-editor

   ```

   ```bash

   ```

## Использование

После запуска введите команду `help` для просмотра списка доступных команд.

### Формат команд

Все команды вводятся в одной строке, параметры разделяются пробелами:

```
add point 10 20
add line 0 0 5 5
add circle 0 0 10
add square 0 0 0 5 5 5 5 0
add oval 0 0 5 3
add rectangle 0 0 0 4 6 4 6 0
list
delete 1
save myfigures.json
load myfigures.json
help
exit
```

### Список команд

| Команда | Описание | Пример |
|---------|----------|--------|
| `add point <x> <y>` | Создать точку | `add point 10 20` |
| `add line <x1> <y1> <x2> <y2>` | Создать линию | `add line 0 0 5 5` |
| `add circle <x> <y> <radius>` | Создать окружность | `add circle 0 0 10` |
| `add square <x1> <y1> <x2> <y2> <x3> <y3> <x4> <y4>` | Создать квадрат (4 вершины) | `add square 0 0 0 5 5 5 5 0` |
| `add oval <x> <y> <radius1> <radius2>` | Создать овал (эллипс) | `add oval 0 0 5 3` |
| `add rectangle <x1> <y1> <x2> <y2> <x3> <y3> <x4> <y4>` | Создать прямоугольник (4 вершины) | `add rectangle 0 0 0 4 6 4 6 0` |
| `list` | Показать все созданные фигуры | `list` |
| `delete <id>` | Удалить фигуру по ID | `delete 3` |
| `save <filename>` | Сохранить фигуры в JSON-файл | `save myfigures.json` |
| `load <filename>` | Загрузить фигуры из JSON-файла | `load myfigures.json` |
| `help` | Показать список команд | `help` |
| `exit` | Выйти из программы | `exit` |

## Особенности реализации

### Валидация фигур
- **Квадрат**: проверяется равенство всех сторон и диагоналей
- **Прямоугольник**: проверяется равенство противоположных сторон и диагоналей
- Все координаты и размеры принимаются как целые числа

### Формат сохранения
Фигуры сохраняются в JSON-формате, что обеспечивает:
- Читаемость файла
- Возможность редактирования вручную
- Совместимость с другими программами

### Обработка ошибок
Программа обрабатывает различные ошибки ввода:
- Неверный формат ID при удалении
- Некорректные параметры фигур
- Ошибки при работе с файлами
